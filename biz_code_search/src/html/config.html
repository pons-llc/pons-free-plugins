法人番号検索プラグイン 設定

経済産業省が提供するgBizINFO REST API v2を使い、「法人番号→法人情報入力」「法人名→法人番号検索」の 2つのボタンをレコード追加・編集画面に設置します。各ボタンの隣には「クリア」ボタンも自動的に 表示され、法人番号・法人名・転記済みの内容をまとめて消去できます。

APIトークン

gBizINFOのWeb API利用申請ページ から取得したAPIトークンを入力してください(申請後、即日〜数時間程度で登録メールアドレスに APIトークンが届きます)。入力したトークンはkintoneプラグインの秘匿領域に保存され、 アプリ利用者やこの設定画面からも再表示できません。既にトークンを設定済みの場合、この欄を 空欄のまま保存すると既存のトークンを維持します。
