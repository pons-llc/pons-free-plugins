複数レコードサブテーブル化プラグイン 設定

別アプリのレコードを検索条件で絞り込み、このアプリのレコード追加/編集画面に設置されるボタンから、 現在のレコードのサブテーブルへ行として取り込みます。実行のたびにサブテーブルの内容は全置換されます (既存行はクリアされ、取得結果で上書きされます)。
