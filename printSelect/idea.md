# 印刷選択プラグイン(printSelect)

## 経緯

すでに本番環境で運用中のプラグインを、本リポジトリの管理下(`<plugin>/src/`構成)に取り込み、
公開サイト(`site/`)で配布するために追加した。ロジックは移行前のまま変更していない
(署名鍵も運用中のものをそのまま使い回しており、プラグインIDは変わらない)。

## 機能

kintoneのレコード印刷画面で、あらかじめ指定しておいたフィールドを非表示にするプラグイン。
帳票のテンプレートに含めたくない項目(内部管理用メモなど)を、印刷時だけ隠したい場合に使う。

## 設定画面

- `kintone.api()`(内部向けラッパー)経由で`GET /k/v1/app/form/fields.json`を呼び出し、
  アプリの全フィールドをチェックボックス一覧として描画する。
- チェックしたフィールドのフィールドコード配列を`kintone.plugin.app.setConfig()`で保存する。

## レコード印刷画面

- `app.record.print.show`イベントで、保存済みのフィールドコード一覧を読み込み、
  `confirm()`で「印刷時に特定のフィールドを隠しますか？」を確認したうえで、
  `Yes`の場合のみ`kintone.app.record.setFieldShown(fieldCode, false)`で非表示にする。

## 実装

外部ライブラリ・外部通信は使用しない。フィールド一覧の取得のみJavaScript APIに相当する
ものがない(`kintone.app.getFormFields()`は現状の実装では未使用)ため、REST API
(`kintone.api()`経由)を使用している。
