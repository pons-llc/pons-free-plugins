プラグイン利用状況AI検索台帳プラグイン 設定

このアプリを「プラグイン台帳アプリ」として使うための設定です。保存すると、台帳に必要なフィールド (プラグインID/プラグイン名称/バージョン/詳細/利用アプリ一覧テーブル)をこのアプリへ自動的に 作成・運用環境へ反映します(数秒かかることがあります)。

簡易AI検索をONにすると、レコード一覧を開いたユーザーのブラウザから外部CDN(cdn.jsdelivr.net)へ 通信が発生します。これは本プラグインに限りCLAUDE.mdの「外部通信を行わない」方針の例外として 認めているものです。詳細はsecurity-checklist.mdを参照してください。ダウンロード機能は AI検索の設定に関わらず常に利用でき、外部通信は発生しません。

「プラグイン利用状況を同期」ボタンを表示するグループ

ここで選んだグループに所属するユーザーにのみ、レコード一覧の同期ボタンを表示します。 ただし実際にプラグイン利用アプリの一覧を取得できるのはcybozu.com共通管理者のみのため、 これはボタンの表示・非表示を切り替えるだけのUI上の絞り込みです(真の権限境界では ありません。security-checklist.md参照)。
