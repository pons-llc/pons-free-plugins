組織ルックアッププラグイン 設定

文字列1行または組織選択フィールドに入力された組織コードをもとに、組織名・説明・親組織の情報を 別の文字列フィールドへ自動入力します。親組織が設定されている場合は親組織の情報も1階層分だけ 取得します(親のさらに親〈祖父にあたる組織〉までは遡りません)。設定行ごとに、次の項目を 設定してください。

  1. ①元フィールド: 組織コードが入力される文字列1行、または組織選択フィールド(組織選択の場合は1つ目の組織のみを使用します)。
  2. ②発動条件: 「ボタン」を選ぶと、指定したスペースフィールドにボタンが設置され、押下時に取得・反映します。「保存時」を選ぶと、レコード保存時に自動で取得・反映します。
  3. ③出力先フィールドを編集可能にするか: オフの場合、追加・編集画面で出力先フィールドを直接編集できなくなります。
  4. ④転記項目: 取得する組織情報の項目と、書き込む出力先フィールド(文字列1行のみ)の組み合わせ。「親組織名」等は、親組織が設定されていない場合は空文字列になります。

元フィールドは、レコード一覧のインライン編集では常に編集できなくなります。該当する組織が 見つからない場合(未入力を含む)、出力先フィールドはすべて空文字列でクリアされます。
