神エクセル帳票印刷プラグイン 設定

Excelテンプレートのセル番地とこのアプリのフィールドを対応付け、レコードの値を差し込んだExcelファイル(xlsx)を 生成・ダウンロードします。テンプレートは、任意のアプリの添付ファイルフィールドにある1件のレコードの ファイル(1番目のファイル)を参照します。

テンプレート参照先

テンプレートを添付したレコードのアプリID・レコードID・添付ファイルフィールドのフィールドコードを指定してください。 フィールドの1番目のファイルをテンプレートとして使用します。

セルマッピング(シート名 + セル番地 ⇔ フィールドコード)

このアプリのフィールドの値を、テンプレートの指定したシート・セルへ書き込みます。

シート名セル番地フィールド

サブテーブル設定(別シートへの繰り返し挿入)

テーブル(サブテーブル)のフィールドを選び、出力先シート・繰り返し開始行・最大行数・列マッピングを設定します。

出力ファイル名

「{フィールドコード}」の形式でこのアプリのフィールド値を差し込めます(例: {customer_name}_請求書)。 拡張子(.xlsx)は自動的に付与されます。

一括ダウンロード

一覧画面での一括ダウンロード(zip)を許可する最大レコード数です。ブラウザ上でファイル生成・圧縮を行うため、 大きすぎる値は動作が重くなる可能性があります。
